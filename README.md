# 🚗 Sistema de Gerenciamento de Concessionária

Este projeto foi desenvolvido como trabalho final do curso técnico no SENAC, em colaboração com meus colegas. Criamos um site para uma concessionária exibir seus carros, com uma página web de design elegante e responsivo.

## 📋 Funcionalidades

- **Cadastro de veículos e clientes**: Gerenciamento completo de informações de veículos e clientes.
- **Consulta e atualização de dados**: Sistema de busca e atualização fácil e rápido.
- **Interface web responsiva**: Página web com design moderno e acessível em qualquer dispositivo.

## 🛠️ Tecnologias Utilizadas

- **Backend**: Django (padrão MTC - Model Template Controller)
- **Banco de Dados**: SQLite
- **Frontend**: HTML, CSS, JavaScript

## 🚀 Como Executar o Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/sistema-concessionaria.git
   cd sistema-concessionaria
   ```

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

3. Execute o servidor:
   ```bash
   python manage.py runserver
   ```
